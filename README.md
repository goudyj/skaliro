# Skaliro

Skaliro is an open source AI learning app concept: a personal coach that can
help someone learn almost anything through conversation, guided practice,
progress tracking, and gamified motivation.

The early product idea is simple:

- connect an AI account or bring your own API key;
- speak or chat with an AI coach;
- define a learning goal such as a language, a programming language, a school
  topic, music, or any concrete skill;
- follow a structured path with exercises, checkpoints, rewards, and a progress
  dashboard;
- keep the platform open enough for many learning domains and community-made
  modules.

## Product Pillars

- AI-first tutoring, not static courses.
- Progress dashboard that shows skills, gaps, streaks, and milestones.
- Gamification with quests, achievements, rewards, and visible progression.
- Open source core with bring-your-own-key support.
- Extensible learning paths for languages, code, maths, music, and more.

## Early MVP

- User onboarding with a learning goal.
- AI chat tutor with a structured session mode.
- First learning path format stored as Markdown or JSON.
- Progress model for lessons, exercises, XP, streaks, and achievements.
- Dashboard with current goal, next actions, and weak areas.
- BYOK support for at least one AI provider.
